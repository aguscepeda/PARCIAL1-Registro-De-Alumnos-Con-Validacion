## DESCRIPCION:

# Aplicacion Web para registrar alumnos mediante un formulario. Con JavaScript cada alumno ingresado debe agregarse dinamicamente solo si LOS DATOS SON VALIDOS. Registros de alumnos eliminables y interfaz mostrando la cantidad total de alumnos cargados. Diseño realizado con Flexbox.

## Tecnologias:

# -HTML5
# -CSS3 (Flexbox)
# -JavaScript (ES6)

## ESTRUCTURA DE PROYECTO:

# 1. assets/: Contiene todos los recursos del sitio (imágenes, vídeos, logos, etc).
# 2. assets/favicon/: Archivos de iconos que aparecen en la pestaña del navegador.
# 3. assets/img/: Todas las imágenes, ilustraciones y vectores necesarios para el diseño.
# 4. css/: Carpeta destinada a las hojas de estilo externas.
# 5. css/components/: Archivos CSS específicos para elementos reutilizables (botones, tarjetas, menús, etc.).
# 6. index.css: Estilos globales y configuración principal de la página.
# 7. js/: Contiene los archivos de JavaScript para la lógica, eventos y manipulación del DOM.
# 8. pages/: En caso de que el proyecto tenga más de una página, aquí se guardarán los otros archivos HTML.
# 9. index.html: El archivo principal y punto de entrada de la aplicación.
# 10. README.md: Documentación del proyecto, integrantes, desarrollo descripción de funcionalidades y guía de instalación si es necesaria.
